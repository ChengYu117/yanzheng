### 1\. Cunningham et al. (ICLR 2024)

#### 核心贡献

这篇工作的重要性在于，它不仅展示了 SAE 可以找到“高可解释特征”，还尝试把这些特征用于更精细的机制定位（mechanistic localization）。

#### 评估方法

1.  **自动解释评分（interpretability score）**  
    先让模型根据高激活上下文生成特征解释，再让模型根据解释预测激活，最后比较预测激活与真实激活的一致性。
2.  **机制干预效果（patching / KL divergence）**  
    用 SAE 特征替代原始表示，观察输出分布与目标分布之间的 KL divergence 是否下降。

#### 指标含义

*   **Interpretability score**：解释文本是否真的对应特征激活模式
*   **KL divergence**：替换表示后模型行为偏离原模型的程度，越低越好

#### 相关数值

*   稀疏编码特征在部分层上的自动解释得分大约在 **0.25–0.30**
*   对比方法（如 PCA / ICA / Random）通常更低，约在 **0.05–0.15**

> 这里原报告中的数值来自图中读数，属于近似值；在正式稿中建议标注“approx.”。

### 2\. Gao et al. (OpenAI 2024)

#### 核心贡献

这是 SAE 走向“规模化”和“标准化评估”的关键工作。

#### 评估方法

1.  **Zero-ablation fidelity**
2.  **Downstream loss / KL**
3.  **Probe suite**
4.  **N2G（自然语言解释到激活预测）的 precision / recall**

#### 指标含义

*   **Zero-ablation fidelity**：插入 SAE 后模型功能恢复程度
*   **Probe loss**：单个 latent 是否足以恢复某个已知特征
*   **Precision / Recall**：解释是否既准确又不泛化过度

#### 相关数值

*   **16M SAE 的 zero-ablation fidelity = 98.2%**
*   Probe baseline score 约为 **0.600**
*   高质量特征示例：
    *   **precision = 0.97**
    *   **recall = 0.56**
*   低质量特征示例：
    *   **precision = 0.06**
    *   **recall = 0.05**

### 4\. JumpReLU SAE (2024)

#### 核心贡献

用可学习阈值和近似 L0 目标改善稀疏-重构权衡。

#### 评估方法

1.  **FVU**
2.  **delta LM loss**
3.  **dead feature ratio**
4.  **人类评审**
5.  **自动解释相关系数（Pearson correlation）**

#### 指标含义

*   **FVU**：未解释方差比例，越低越好
*   **delta LM loss**：插入 SAE 后语言模型损失增加多少，越低越好
*   **dead features**：几乎从不激活的特征占比
*   **Pearson r**：解释驱动的激活预测与真实激活的一致性

#### 相关数值

*   原始 gated loss 在 layer 20 上可能出现约 **40% dead features**
*   JumpReLU / TopK 通常 dead features 明显更低，图中接近 **0–3%**
*   自动解释相关系数约在 **0.4–0.5**
*   随机初始化基线约在 **0.2**
*   人类评审中 “Yes” 比例约 **0.65–0.72**

### 9\. Progressive Coding (2025)

#### 核心贡献

比较多尺度 SAE 的不同构造方式。

#### 评估方法

1.  **FVU**
2.  **recaptured LLM loss**
3.  **RSA**
4.  **自动解释 Pearson correlation**

#### 指标含义

*   **recaptured LLM loss**：插入 SAE 后恢复原模型行为的程度
*   **RSA**：原始激活与重构激活在表征几何上的相似度
*   **Pearson r**：解释一致性

#### 相关数值

*   幂律拟合指数：
    *   ** $\alpha \in [-0.54, -0.72]$ **
    *   ** $R^2 \in [0.916, 0.922]$ **
*   FVU 与 recaptured 指标相关性约 **0.8**
*   自动解释 Pearson 均值约 **0.57–0.74**

### 10\. End-to-End Sparse Dictionary Learning (2024)

#### 核心贡献

不再只优化激活重构，而是直接优化输出 logits 分布。

#### 评估方法

1.  **CE increase**
2.  **L0**
3.  **alive features**
4.  **下游层重构误差**
5.  **logits KL**

#### 指标含义

这个方向强调：  
**解释性特征必须是“功能重要的”，而不只是“重构友好的”。**
